## How to use it
1. git clone this repo
2. install virtualenvironment for python3
3. install all required dependencies listed in requirements.txt
4. run client.py with python3 on the Raspberry Pi
5. run server.py with python3 on any server
6. read client.py to see which GPIO pins are used 
7. set up the circuit 
8. position the camera such that there are no shadows cast over the black digit on the white background
9. press the button to take a picture
10. verify classification
